# Install and Run Project

***clone the repository***

```bash
git clone https://github.com/atharvahande11/NSP_Winter_Internship.git
```


***change to the project directory***

```bash
cd NSP_Winter_Internship/myapp

```
***Install Vite***

```bash
npm install vite --save-dev
```

***Run server***

```bash
npm run dev
